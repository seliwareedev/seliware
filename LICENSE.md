## 🔍 Troubleshooting Common Problems

### ❌ "Failed to Attach" Error

**Cause:** Roblox may not be fully loaded yet, or a conflicting process is running.
**Fix:**
1. Wait for the Roblox game to fully load (past the loading screen)
2. Close any other injection tools that may be running
3. Restart Seliware as Administrator on Windows
4. Increase `injectDelay` in your config to `1000` or higher

### ❌ Antivirus Blocking the Download or Launch

**Cause:** Heuristic detection of injection behavior — this is a false positive.
**Fix:**
1. Add the Seliware installation folder to your antivirus exclusion list
2. Temporarily disable real-time protection during installation
3. Re-enable protection after installation is complete

### ❌ Script Doesn't Execute (No Error Shown)

**Cause:** Script may be incompatible with the current engine version, or attachment failed silently.
**Fix:**
1. Re-attach manually using the Attach button
2. Check the execution log for hidden errors (View → Logs)
3. Try a known-working script from the hub to confirm the engine is functional

### ❌ Mobile App Crashes on Launch

**Cause:** Insufficient RAM or overlay permission not granted.
**Fix:**
1. Close background apps to free RAM
2. Go to **Settings → Apps → Seliware → Permissions** and enable Display Over Other Apps
3. Reinstall if the issue persists

### ❌ iOS App Disappears After 7 Days

**Cause:** Free Apple ID certificate expiry — this is a limitation of sideloading.
**Fix:** Re-sideload using your tool of choice. The process takes under 2 minutes once you've done it once.
